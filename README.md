# 3. vim設定
## Icebergインストール

``` zsh
$ mkdir -p ~/.vim/colors
$ cd ~/.vim
$ git clone https://github.com/cocopon/iceberg.vim
$ cp iceberg.vim/colors/iceberg.vim colors/.
```
